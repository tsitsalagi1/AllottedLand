How to use the Workbench

  1. Select the LOC page / township / range. Page 29 is currently the Cherokee Nation atlas page for T24N R14E.
  2. Load the full source map image. Do not upload source images to the website; the file stays in your browser.
  3. Calibrate the grid. Move the outside blue lines to the outside township/range section grid, then distribute and adjust internal lines.
  4. Pick a section. The section zoom is generated live from the full image.
  5. Enter exactly what you can read. Use First, Middle, and Last name fields. If there is no middle name or initial, leave Middle blank.
  6. Do not guess number types. Put the visible number in “Number shown on map” and leave the type as unknown/map number unless separately verified.
  7. Submit queued rows. A trusted reviewer key sends rows directly to approved records. Without a key, rows go to pending admin review.

How to read the LOC map labels before entering a record
  • Names: enter the person's name as read from the map. If there is no middle name or middle initial, leave Middle blank.
  • Numbers beside names: enter the number exactly in Number shown on map. Do not force it into Roll # or Allotment # unless you know what it is.
  • Status letters: small notes such as H, minor, or other marks go in Status/restriction note exactly as shown. Do not guess what an unclear mark means.
  • Dawes/NARA fields: Roll, enrollment, and census-card numbers should stay blank unless verified from Dawes/NARA or another source outside this map.
  • Uncertainty: if spelling, middle initial, or number type is uncertain, choose confidence needs-review or uncertain and explain in Notes.
Example: Map reads “Sarah R. Gourd” with number “18338.” Enter First = Sarah, Middle = R., Last = Gourd, Number shown on map = 18338, Number type = unknown/map number unless separately verified.
The LOC item describes this atlas as showing names, statuses, and allotment numbers, but this tool preserves the exact map number separately so reviewers do not accidentally mislabel roll/enrollment/census-card numbers.
