Help index the maps

The production workflow is now human-reviewed full-map transcription. OCR candidates are deprecated.

Current workflow

  1. Open the Full Map Workbench. Use tools/map_workbench.html.
  2. Load one full LOC source image. The image stays in the browser session and is not stored by the website.
  3. Calibrate the township/range grid. Set the outside grid first, distribute internal lines, then adjust any line that is off.
  4. Select a section. The tool zooms into that section from the full map image.
  5. Human-enter records. Enter first, middle, and last name; number shown on map; status note; section; source link; and confidence.
  6. Export approved rows. Merge reviewed rows into data/allotment_records.json.
  7. Track section progress. Mark each section complete, needs review, no records, or not started.

Deprecated tools

The OCR agent, candidate review page, section-crop workflow, and township folder workbench were experiments. They helped us learn the right structure, but they are no longer the production path.

Important data-entry rule

Do not force a number into roll number, enrollment number, census-card number, or allotment number unless the source supports it. If the map only shows a number beside a name, enter it as number_shown_on_map / map_number and leave the specific Dawes/NARA fields blank until separately verified.
