Volunteer indexing

Help index the maps

This page is for people helping the project turn old map images into reviewed searchable records.

For regular family research, start with Search

If you are trying to find family land records, you probably want the Universal Search, not this volunteer page.

Use the Search link in the header for family research.

What volunteers do

  1. Open an original source map.
  2. Read names, numbers, and section information from the map.
  3. Enter only what appears on the historical public source.
  4. Mark uncertainty instead of guessing.
  5. Submit rows for review before they become public search results.

Do not add private family information

Map indexing should use source-linked historical records. Do not add private information about living people, family rumors, or unsupported legal claims to a public map row.
