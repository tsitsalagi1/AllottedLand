Transcribe Records

Help build the AllottedLand.com records index section by section.

Primary tool

Use the full map workbench to load one LOC source image, calibrate the grid, zoom into a section, enter human-reviewed rows, and submit them to the website database.

Open Map Workbench

The source map image stays in your browser. Only reviewed row data and section status are submitted.

Setup note

Website submission requires the Cloudflare D1 backend added in v0.26. Until the backend is connected, the workbench will still display and queue rows but submission will show a setup error.
