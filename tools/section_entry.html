Section Entry Workbench v0.18

Human-first section indexing. Load all section crops from one township/range map, pick the section, enter rows from the tile, prevent duplicate entries, and mark each section as complete, needs review, or no records found. This tool runs locally and does not upload anything.

1. Load section crops and existing data

Ready. Start by loading data/ocr_runs/page_###_sections_manifest.json, or manually choose one section image.

How to create all 36 section crops from one LOC map:
python tools\map_indexing_agent.py --page 29 --mode sections --sections all --preprocess soft --section-padding 90 --crops-only
Use --grid-pct if the section boxes are off. Use soft first for human reading; use threshold or invert only if they make the writing clearer.

2. Pick township/range/section and track progress

Standard PLSS section order is shown. Use the grid as a work tracker, not as proof of legal title. Confirm the crop boundaries visually before entering rows.
