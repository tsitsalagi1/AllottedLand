Current focus

The site is being simplified around one main tool: Universal Search. The goal is to help families start anywhere, build a research path, and print a packet without needing to understand every record system first.

Recent changes

  • v0.50: Public pages simplified so each page has one clear purpose.
  • v0.49: Homepage simplified and Universal Search made field-first.
  • v0.48: “Clue” wording changed to “information you have.”
  • v0.45-v0.47: Universal Search, research path, print packet, and agency request packets improved.
