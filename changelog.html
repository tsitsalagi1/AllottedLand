AllottedLand.com · Project history

Project updates

A public change log so families, volunteers, and researchers can see what changed and what is coming next.

Current version: v0.7

AllottedLand.com is still in Beta / Phase 1. The public site currently indexes Library of Congress Cherokee Nation atlas map pages and explains the research process. Verified public name, roll-number, allotment-number, testimonial, and land-loss datasets are not open yet.

This page is a plain-language project update. For the technical file history, see the GitHub repository commit history.

v0.8 — Map Indexing Agent starter kit

  • Added a local OCR agent for testing one LOC map page at a time.
  • Added candidate-row output in data/allotment_records_candidates.json.
  • Added a local review helper so OCR candidates can be checked before publishing.
  • Added OCR workflow documentation and Python requirements.
  • Kept OCR results as unverified leads only; human review remains required.

v0.7 — Project updates and version tracking

Added June 2026

  • Added this Project Updates page.
  • Added CHANGELOG.md for GitHub version tracking.
  • Added project-update links in navigation/footer areas.
  • Updated the sitemap so search engines can discover the updates page.
  • Kept the upload/submission system closed until privacy, consent, and review rules are finalized.

v0.6 — Contact button fix

  • Replaced primary contact buttons with Gmail web-compose links.
  • Kept a default email-app fallback for visitors who prefer another email app.
  • Added a copy-email button for allottedland@gmail.com.

v0.5 — Project contact email

  • Added allottedland@gmail.com across the site.
  • Updated the Contact page for questions, corrections, removal requests, and volunteer indexing.
  • Added reminders not to email sensitive private family documents unless redacted and permitted.

v0.4 — Data and search improvement

  • Added Phase 1 data-status language.
  • Added the Help Index Maps page.
  • Added a safe transcription workflow and templates.
  • Improved map preview behavior.

v0.3 — Trust and source pages

  • Added About, Source Records, and Contact pages.
  • Added sitemap and robots.txt updates.
  • Added Beta / Phase 1 language and footer links.

v0.2 — Privacy fix and search cleanup

  • Removed personal family sample data from the public example text.
  • Changed the search page so it starts with no results until a user searches.
  • Clarified map transcription/indexing status.

v0.1 — Public foundation

  • Created the first public static site foundation.
  • Added homepage, guided finder, search prototype, request builder, privacy, terms, and submission consent pages.
  • Started with the Library of Congress Cherokee Nation atlas map index.
