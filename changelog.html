AllottedLand.com · Project history

Project updates

A public change log so families, volunteers, and researchers can see what changed.

v0.25 — Cleanup and production-tool focus

  • Made the Full Map Workbench the active production tool.
  • Deprecated the OCR agent, candidate review page, crop-folder workflow, and older local helper pages.
  • Added a cleanup plan and .gitignore rules to keep generated images, OCR output, and temporary exports out of the repository.
  • Kept the repo focused on verified records, section progress, source links, and public website files.

v0.24 — Data-entry guide and safer map-number fields

  • Added map-label guidance for new reviewers.
  • Added first/middle/last name fields and safer number_shown_on_map / number_type fields.

v0.23 — Full Map Workbench

  • Added one-image workflow: load full LOC map, calibrate grid, zoom selected section, enter records, export JSON.
