Mission

Start with what you know. Follow the record trail.

AllottedLand.com helps Native families turn family information into a documented research path. A name, spelling variant, township/range, county, cemetery, old town, map number, or family story can become the starting point for finding source records.

The site is built around original records: historical allotment maps, enrollment records, allotment jackets, county land records, BIA/LTRO title records, tribal records, NARA records, and family-submitted evidence that has been reviewed before publication.

One tool, one path

Read this first: one search box builds the whole research path.

Families should not have to know which form or agency to start with. Put the information you have in the Universal Search below. The site will build the research path first, then show matching site records, official-source leads, geography leads, and agency request packets.

Advanced source connectors still run behind the scenes when they are useful. The home page now keeps the user focused on one plain-English workflow.

How to read the Universal Search results

  1. Built research path comes first. This is the roadmap. It tells the family what record to look for, where to search, and why the record matters.
  2. Matching site records come second. These are records already loaded into AllottedLand.com. If nothing matches yet, that does not mean no record exists; it means the site index is still being built.
  3. Official source leads come next. These cards point users to NARA, OHS, LOC maps, Federal Register, historic newspapers, Census/TIGERweb geography, BIA/LTRO, and BLM/GLO paths.
  4. Agency request packets are last. Copy the request for the agency or county office. Each request is written in plain English so the office understands what the user is asking for.

What the family should collect

  • Ancestor name, spelling variants, relatives, and tribe/Nation
  • Dawes roll/enrollment number and census card number
  • Enrollment application/testimony packet
  • Allotment jacket or application for allotment
  • Legal description: township, range, section, acreage, county
  • Allotment map page or source image
  • County deed, mortgage, release, sheriff deed, tax deed, probate, guardianship, or oil/gas record
  • BIA/LTRO title records or Title Status Report path if trust/restricted land may be involved
